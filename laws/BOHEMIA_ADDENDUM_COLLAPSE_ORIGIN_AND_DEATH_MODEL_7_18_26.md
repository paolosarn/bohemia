## LOCKED THIS SESSION (Paolo asserted these firmly)

1. THE COLLAPSE WAS MULTI-CAUSAL. The economic apocalypse was not one event. It
   was several correlated crises popping at the same time (correlated, not
   coincidental: connected systems fail together). This supersedes any reading
   that treated the collapse as a single trigger.

2. THE IMMORTALITY / RE-EMBODIMENT TECH BUBBLE IS A CONFIRMED PILLAR. The
   economy over-invested in buying the few out of death (the download / getting
   a mind back into a body was the holy grail everyone chased; the upload,
   capturing the mind, was the hard part). The Amalgamation is the ruins of that
   moonshot. This confirms the origin proposed earlier the same day.

3. AUTHORIAL INTENT: THE BACKSTORY IS A WARNING, NOT A CLAIM ABOUT HUMAN NATURE.
   Paolo, verbatim in spirit: "this isn't necessarily how I think human beings
   are, but it is supposed to be a warning." Every design decision downstream of
   the collapse serves the warning, never cynicism for its own sake.

4. DEATH MODEL: NOT PERMADEATH (corrects a misread). Ordinary player death
   reloads the nearest save point. The game does NOT restart from the beginning
   on death. The "roguelite" descriptor in older files refers to THE FOLD, the
   generational dynasty structure across the 100-year arc (a dynast's death
   advances the line), NOT a one-life-and-restart challenge. Any file implying
   permadeath is superseded by this line.

## THE RECONSTRUCTION RULE (strongly implied by locked canon; [PENDING] explicit lock)

Locked canon (GDD v5): the Amalgamation reads the RECORDED dead (corpses log as
persistent world memory). That is reconstruction from traces after death, NOT a
live 100%-fidelity scan. Consistent consequence, which Paolo built on this
session: the Amalgamation can only rebuild the RECORDED self. Everything lived
off the ledger, everything never posted to the feed, is absent from the
reconstruction. The ghost that returns is the performed / curated / feed self,
with the unrecorded soul missing. This fuses the social feed, the
recorded-vs-unrecorded self (the single most-ported theme across the 150
quests, 66 times), and the coin-toss lie into one mechanism. Mark as the
working reading; awaiting Paolo's explicit lock.
